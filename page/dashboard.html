เทมเพลตแอดมิน ?

เทมเพลตแอดมิน – ถูกพัฒนาขึ้นโดย JiMeBillie เป็นแหล่งทรัพยากรสำหรับ "นักพัฒนาเว็บ" ที่ถูกพัฒนาด้วยภาษา HTML, CSS และ JavaScript ทำให้นักพัฒนาที่นำไปใช้ สามารถโฟกัสกับการพัฒนาเนื้อหาหลักได้อย่างเต็มที่ โดยไม่จำเป็นต้องเสียเวลาในการออกแบบหน้า "Dashboard" ขึ้นมาใหม่แต่ละครั้ง

"เทมเพลตแอดมิน" นี้ ทำให้การเริ่มต้นในการสร้างเว็บแอพพลิเคชันหรือแดชบอร์ดเว็บใหม่นั้นง่ายและประหยัดเวลามากขึ้น ผู้ใช้สามารถใช้เทมเพลตแอดมินเพื่อทำให้ส่วนของการสร้างโครงสร้างและการจัดวางหน้าเว็บเป็นเรื่องง่าย ทั้งนี้ยังทำให้ผู้ใช้สามารถใช้เวลาในการพัฒนาส่วนประกอบและฟีเจอร์หลักของเว็บแอพพลิเคชันและเว็บไซต์ได้มากขึ้น

การพัฒนาเว็บแอพพลิเคชันและเว็บไซต์ที่ดี "เทมเพลตแอดมิน" จาก "JiMeBillie" ช่วยให้การสร้างและพัฒนาแอพพลิเคชันหรือเว็บไซต์ เป็นเรื่องที่น่าสนใจ เพื่อให้ผู้ใช้สามารถนำไปใช้งานและพัฒนาต่อได้อย่างมีประสิทธิภาพและสมบูรณ์

เวอร์ชั่น

ณ ปัจจุบัน เทมเพลตแอดมิน ถูกพัฒนาโดย "JiMeBillie" ซึ่งยังไม่มีทีมสำหรับพัฒนา ทำให้การปล่อยเวอร์ชั่นเป็นไปอย่างล่าช้า แผลนของการปล่อยเวอร์ชั่น คือ จะปล่อยเวอร์ชั่นถัดไปเมื่อพัฒนาเพจ หรือ Feature ใหม่เสร็จ – หากมีความสนใจอยากร่วมทีมพัฒนา สามารถ ขอร่วมทีมพัฒนา

วิธีการใช้งานเทมเพลต

ก่อนอื่นดาวน์โหลด Template ไฟล์ .zip : ดาวน์โหลด

  • 1. แตกไฟล์ template-admin.zip
  • 2. เปิด Folder template-admin จากนั้น เปิดไฟล์ "index.html" ด้วย Browser
  • 3. "Sidebar" ด้านซ้ายจะมีเมนูให้เลือกหน้าเพจที่ต้องการ เช่น ต้องการ "หน้าล็อกอิน" ให้คลิกเลือกที่ เมนูหน้าล็อกอิน
  • 4. เมื่อได้หน้าที่ต้องการ ให้คลิกขวา แล้วเลือก "View page source" จากนั้น คัดลอก Source code ไปใช้งานได้เลย

ป.ล. ไฟล์ทรัพยากรที่สำคัญ คือ "environment-jimebillie" ซึ่งรวมไปด้วยไฟล์ CSS , JavaScript และ Plugin อื่น ๆ ซึ่งจุดสังเกตในแต่ละหน้าที่ต้องการใช้งานคือ "Tag" link rel="stylesheet" href=" " ที่อยู่ใน Tag Head และ script src=" " ที่อยู่ ส่วนท้ายสุด ของ Source Code ซึ่งข้อควรระวัง ควรคำนึงถึง Path ตำแหน่งที่อยู่ของ Folder "environment-jimebillie" เพราะเชื่อว่าแต่ละ Project อาจจัดวาง Path ของ Folder แตกต่างกันไป

หน้าล็อกอิน เวอร์ชั่น 1

หากมีความต้องการเปลี่ยนรูปภาพ ของหน้าเพจล็อกอิน ให้เปลี่ยนที่ส่วนล่างของ Tag Script ที่อยู่ด้านล่างของ Source code ในส่วนของ Object ที่ชื่อ : "change_cover_image" โดยคัดลอกที่อยู่ภาพมาวางได้เลย
