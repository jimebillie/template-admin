เทมเพลตแอดมิน ?

เทมเพลตแอดมิน – ถูกสร้างขึ้นโดย JiMeBillie เป็นแหล่งทรัพยากรสำหรับ "นักพัฒนาเว็บ" แม้ว่าจะถูกพัฒนาด้วยภาษา HTML, CSS และ JavaScript ก็ตาม ทำให้นักพัฒนาที่นำไปใช้ สามารถโฟกัสกับการพัฒนาเนื้อหาหลักได้อย่างเต็มที่ โดยไม่จำเป็นต้องเสียเวลาในการออกแบบหน้า "Dashboard" ขึ้นมาใหม่แต่ละครั้ง

"เทมเพลตแอดมิน" นี้ ทำให้การเริ่มต้นในการสร้างเว็บแอพพลิเคชันหรือแดชบอร์ดเว็บใหม่นั้นง่ายและประหยัดเวลามากขึ้น ผู้ใช้สามารถใช้เทมเพลตแอดมินเพื่อทำให้ส่วนของการสร้างโครงสร้างและการจัดวางหน้าเว็บเป็นเรื่องง่าย ทั้งนี้ยังทำให้ผู้ใช้สามารถใช้เวลาในการพัฒนาส่วนประกอบและฟีเจอร์หลักของเว็บแอพพลิเคชันและเว็บไซต์ได้มากขึ้น

การพัฒนาเว็บแอพพลิเคชันและเว็บไซต์ที่ดี "เทมเพลตแอดมิน" จาก "JiMeBillie" ช่วยให้การสร้างและพัฒนาแอพพลิเคชันหรือเว็บไซต์ เป็นเรื่องที่น่าสนใจ เพื่อให้ผู้ใช้สามารถนำไปใช้งานและพัฒนาต่อได้อย่างมีประสิทธิภาพและสมบูรณ์

ผังเวอร์ชั่น

ณ ปัจจุบัน เทมเพลตแอดมิน ถูกพัฒนาโดย "JiMeBillie" เพียงผู้เดียว ซึ่งยังไม่มีทีมสำหรับพัฒนา ทำให้การปล่อยเวอร์ชั่นเป็นไปอย่างล่าช้า แผลนของการปล่อยเวอร์ชั่น คือ จะปล่อยเวอร์ชั่นถัดไปเมื่อพัฒนาเพจ หรือ เมนูเสร็จ – หากมีความสนใจอยากร่วมทีมพัฒนา ติดต่อ อีเมล : forwork.jirapat.meedoung.billie@gmail.com

วิธีการใช้งานเทมเพลต

ก่อนอื่นดาวน์โหลด .zip หรือ Clone จาก Repository ของ Github นี้ : template-admin

  • 1. เปิด Folder template-admin จากนั้น เปิดไฟล์ "index.html" ด้วย Browser
  • 2. "Sidebar" ด้านซ้ายเลือกหน้าเพจที่ต้องการ เช่น ต้องการหน้าล็อกอินให้เลือกที่เมนูหน้าล็อกอิน
  • 3. เมื่อได้เพจที่ต้องการแล้วให้คลิกขวา แล้วเลือก "View page source" จากนั้น คัดลอก Source code ไปใช้งาน

ป.ล. ไฟล์ทรัพยากรที่สำคัญอย่าลืม Folder "environment-jimebillie" ซึ่งรวมไปด้วยไฟล์ CSS , JavaScript, Font, และไฟล์ภาพ

ดาวน์โหลดเพจ HTML ดาวน์โหลดเพจ CSS ดาวน์โหลดเพจ JS

หากมีความต้องการเปลี่ยนรูปภาพ ของหน้าเพจล็อกอิน ให้เปลี่ยนที่ส่วนล่างของ Tag Script ที่อยู่ด้านล่างของ Source code ในส่วนของ Object ที่ชื่อ : "change_cover_image" โดยคัดลอกที่อยู่ภาพมาวางได้เลย
